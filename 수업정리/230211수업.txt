- 함수 종류 : 내장 함수, 사용자 정의 함수
- 함수 정의 및 호출
- Java Script : 인터프리터 언어(interpreter) - 대화형 언어
- 중첩 함수, 콜백 함수
- 클로저
- 스코프(scope) : 함수 레벨 스코프, 블록 레벨 스코프, 정적 스코프(렉시컬 스코프)
- 변수 종류 : 전역 변수, 지여 변수, 멤버 변수
- DOM을 이용한(통한) element 접근

내장 함수 : 만들어져 있는 함수
사용자 정의 함수 : 만들어서 쓰는 함수 (사용자는 개발자)
JS는 함수 안에 함수 정의를 할 수 있다, 이것이 중첩함수
스코프는 범위, 영역 이라고 한다
